# **Advent of Cyber 2025**

Welcome to my personal writeups for **TryHackMe’s Advent of Cyber 2025** — a free, beginner-friendly cybersecurity event offering 24 days of hands-on challenges, spanning offensive, defensive, cloud, malware, forensics, and AI-security topics.

This repository documents solutions, notes, and learning outcomes for each day.

---

# **Event Overview**

* A 24-day guided cybersecurity challenge running from **December 1st to December 24th**.
* Designed for **absolute beginners to intermediate learners**.
* Includes **certificates, daily tasks, hints, videos, and industry mentor guidance**.
* Covers topics ranging from **Web Security to Incident Response to Malware Analysis to AI Security**.
* Features **community leaderboards, team challenges**, and a storyline centered on the frozen town of Wazaville.
